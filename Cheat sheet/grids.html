Commands and elements for Grids

Grids

frUnit of measurementBy using the fr unit, we can define the size of columns and rows as a fraction of the grids length and width.
Tag Name Description
grid-Template-rows justifys the row and colums of the grid
grid-Template-area examples: grid-template-columns:12p,12px,12px; grid-template-rows:12p,12px,12px; Define the row and columns of the grid
justify-items examples: justify items: center; Aligns content in a grid item along the rows axis
align-items example: align-items:strech; Aligns content in a grid item along the colums axis
justify-content example: justify-content:center; justfies all grid content on rows axis when total grid size is smaller then container
align-content example: align-content:center; justfies all grid content on column axis when total grid size is smaller then container
grid-auto-flow example: grid-auto-flow:dense; Alogrithm for automatically placin grid items that aren't explictly placed.
row-gap/column-gap/gap example: grid-gap: 1px 9px; Specifies the size of the colums and row gutters.
grid-row-start/ grid-row-end example:grid-row-start:1; grid-row-end:3; Determines an items row-based location within the grid
grid-column-start/ grid-column-end example: grid-column-start:1; grid-column-end:3; Determines an items column based location within the grid
Justify-self justify-self:strech; aligns content for a specific grid item along the row axis.
Align-self example: align-self:center; aligns content for a specific grid item along the column axis.
