Css Media quiries

Tag Description
@media only screen and (max-width: 480px) CSS uses media queries to adapt a websites content to different screen sizes. With media queries, CSS can detect the size of the current screen and apply different CSS styles depending on the width of the screen.
@media This keyword begins a media query rule and instructs the CSS compiler on how to parse the rest of the rule.
only screen Indicates what types of devices should use this rule. In early attempts to target different devices, CSS incorporated different media types (screen, print, handheld)
and (max-width : 480px) This part of the rule is called a media feature, and instructs the CSS compiler to apply the CSS styles to devices with a width of 480 pixels or smaller.
You can also have other things like min-width or height. also you can use other forms of measurements like dpi. another feature is (orientation:..like landscape ect).
Why use this? CSS rules are nested inside of the media querys curly braces. The rules will be applied when the media query is met.
