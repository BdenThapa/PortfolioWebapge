Table Tags/Attributes

Commands and Tags for Tables/Attributes

Table Tags

Tag Name Description
< table > Table The wrapper element for all HTML tables.
< thead > Table Head The set of rows defining the column headers in a table.
< tbody > Table Body The set of rows containing actual table data.
< tr > Table Row The table row container.
< td > Table Data The table row container.
< tfoot > Table Foot The set of rows defining the footer in a table.
< Button > Button creates an button with the stadard element.

Attributes(css)

Attribute Name Description
colspan Colum Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.
table border table border sets the width of the border around table cells
table width table width sets the width of the table in pixels or as a percentage
tr allign tr allign sets the allignment for cells within rows
td align td allign sets the allignment for cells
tr valign tr valign sets the vertical alignment for cells within a row
td valign td valign sets the vertical alignment for cells
td nowrap td nowrap prevents lines within a cell from being broken to fit
table cellspacing table cellspace sets the amount of space between table cells
table cellpadding table cellpadding sets amount of space between a call's border amd its contents
